# 👥 User Management System

<p align="center">
  <img src="docs/demo.gif" alt="Demonstração do sistema" width="700">
</p>

Aplicação de console desenvolvida em **C#** para gerenciamento de usuários, com operações de cadastro, consulta, atualização e remoção de registros, utilizando persistência de dados em **JSON**.

## ✨ Funcionalidades

- ➕ Cadastro de usuários
- 📋 Listagem de usuários
- 🔍 Busca por matrícula
- ✏️ Atualização de dados
- 🗑️ Remoção de usuários
- 💾 Persistência em arquivo JSON
- ✅ Validação de entradas

## 🛠️ Tecnologias

- C#
- .NET
- Newtonsoft.Json
- System.IO

## 📚 Conceitos praticados

- Programação Orientada a Objetos
- Serialização e desserialização de JSON
- Manipulação de arquivos
- Collections
- Enum
- Organização de código em múltiplas classes
- Validação de dados

## 🚀 Executando o projeto

```bash
git clone https://github.com/rafaelvassis/user-management-system-csharp.git

cd user-management-system-csharp/user-management-system-csharp

dotnet restore

dotnet run
```
